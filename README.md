## Portafolio

* Hecho en Django Rest Framework junto con React en Webpack.
* Se requiere una base de datos sqlite en la carpeta raíz llamada db.sqlite3
* Se pueden modificar en la sección /admin con us respectivo superusuario
* Para reflejar los cambios en la interfaz de usuario se deben ejecutar los siguientes comandos de npm en la carpeta raíz

```bash
npm run install
npm run dev
```
